muse_tools.diagnostics.plot_optics_sheet(configs, figsize=(14, 12), lw=2)[source]
TRANSPORT-style optics sheet with 4 panels:

  1. Beta functions (βx, βy) + Dx

  2. Dispersion-inflated envelopes σx_total and geometric σy

  3. R12, R34 accumulated transport matrix elements

  4. Kx, Ky focusing strengths (piecewise constant)

Uses the same input structure as plot_overlaid_envelopes(configs).

Module contents

muse_tools: Utilities for building and analyzing the MUSE beamline.

This package provides: - Beamline construction (beamline.py) - Optics and sigma propagation (optics.py) - Diagnostic plots (diagnostics.py) - Histogram utilities (histograms.py)
